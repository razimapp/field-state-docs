Fieldstate · dream shell · design direction

Solid is what happened.
Dashed is the dream.

One rule carries the whole identity: recorded football is drawn solid in chalk; modelled estimates glow amber with confidence attached; anything hypothetical — an edited pass, a simulated press, an unplayed future. is violet and dashed, everywhere, always. The interface cannot overclaim because the language won't let it.

Release gate
candidate model→ held-out matches→ pre-registered targets→ GATE: NOT PASSED

Until a gate passes, its model appears nowhere else in the app. The Lab runs on gated-off research? No. the Lab runs on the last release that passed. That sentence is the product's spine.

Provenance ledger

Every number in every room resolves to a row here: source data → transformation → model version → evidence artifact → commit. Recorded Derived Estimate Hypothetical are not styling. they are ledger classes, and the UI reads them from the data contract.
