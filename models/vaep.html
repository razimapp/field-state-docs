Action model

VAEP values each on-ball action by its impact on the game.

It is a strong fit when the question is about the value of the action itself, not just the shot outcome.

Overview Models Data Roadmap

How it works

Estimate how an action changes the chance of scoring or conceding in the next sequence.

Where it helps

Player action ranking, pass value, carry value, press resistance, and tactical review.

Scenario fit

Very strong for editing one action and comparing the value of the original versus the alternative.

How to test it

Check correlation with future goals, future xG, and possession success. Test against a baseline model.

What it misses

It is still an estimate. It does not fully know the hidden tactical intent of a team.

Investor wording

“VAEP helps us measure whether a football action helped the team.”
