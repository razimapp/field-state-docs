Three options

Choose the product before choosing the model.

Score prediction, world simulation, and event-level what-if analysis are all valid paths. They are not the same product.

Path A

Score prediction

Best for live probability, odds comparison, and tournament forecasting.

Strength: easier to explain.
Limit: crowded market.

Path B

Football world simulation

Best for clubs, leagues, transfers, and long-term outcomes.

Strength: product depth.
Limit: less direct to match decisions.

Path C

Event-level what-if engine

Best for analysts, coaches, academies, and tactical scenario planning.

Strength: most differentiated.
Limit: harder to build well.

Recommendation

Start with Path C for the analyst MVP, and borrow the clarity and polish of Path A and Path B.
