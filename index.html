Fieldstate

What if a football decision changed the next phase of play?

Fieldstate is a simple way to explore football scenarios. It compares decisions, values on-ball actions, and shows how the next few events may change.

The first product is for analysts. The longer path can include clubs, academies, media, and live match storytelling.

See product paths See the models Print MVP proposal

The core idea

Tools today usually show what happened. Fieldstate is about what could happen next. That makes it useful for tactical review, player decision-making, and scenario planning.

Simple

Change one action

Adjust a pass, carry, shot, or turnover and compare the likely next phase.

Practical

Use real football data

Train on event data, lineups, and freeze frames so the output stays grounded.

Useful

Help people decide

Show analysts and coaches why one option may be better than another.

Best first step

Start with an analyst MVP. Let a user pick a real event, edit the action, and see the impact on possession value, chance creation, and turnover risk.

That is easier to explain, easier to test, and easier to sell than a full alternate-reality match replay.

Fieldstate is a working name for the project.
