What exists now

Different tools solve different parts of the problem.

Some tools simulate matches. Some value actions. Some train predictive models. Fieldstate can sit between them and connect the pieces.

Project What it already does Best fit What is missing
open-football Strong UI, player pages, match pages, world simulation. Score prediction and football world views. Event-level what-if branching and action editing.
footballSimulationEngine Iterative match engine with editable player state and actions. Scenario sandbox and match engine ideas. Real-data calibration and modern analytics layer.
socceraction SPADL conversion, xT, VAEP, action value models. Analyst-grade event valuation. Interactive scenario UI and branching simulation.
StatsBomb Open Data Free event data, lineups, and some 360 freeze-frame data. Training and demos. Commercial breadth and full coverage.
AI-WC-Football Prediction stack, live odds, Monte Carlo, calibration. Score prediction and market-style outputs. Event-level action simulation.
