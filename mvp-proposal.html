Football What-If Simulation MVP

A first-pass MVP to test whether a football event can be adjusted and followed by a plausible alternative next phase of play.

Working title only. Candidate names include NextPhase, FieldState, and MatchMind. Final naming to be decided by investors.
Objective

Show what could happen next

The MVP lets a person select one event from a real football match, adjust it, and view a plausible alternative next phase of play.

Scope

Keep the first version narrow

The first version should simulate the next phase, not claim to replay the whole match or predict a perfect alternate reality.

MVP Scope

Included in the first pass

  • StatsBomb Open Data loading
  • Real match and event selection
  • Pass or carry edit flow
  • Original versus counterfactual comparison
  • Basic next-phase simulation
  • 2D pitch visualisation
  • Action-value output using xT / VAEP-style logic
  • Short feasibility report

Excluded from the first pass

  • Full-match alternate reality simulation
  • Paid data integration
  • Live betting product
  • Elite-club-grade player-style modelling
  • Full tracking-data realism
  • Large multi-user SaaS platform
  • Transformer training from day one
  • Commercial accuracy claims

30 Working Day Plan

Week 1

Load data, show matches, browse sequences, and select events.

Week 2

Convert events to SPADL, add action value, and support pass/carry edits.

Week 3

Build baseline next-phase simulation and branch comparison.

Week 4-6

Polish 2D visualisation, test plausibility, and prepare the investor demo.

Success for the first pass means the output is understandable, plausible, and useful enough to justify deeper data access and model investment.

Use of Anthropic Fable

Where Fable helps

Fable is best used as a build accelerator for architecture, coding support, debugging, evaluation design, documentation, and investor materials.

Where Fable should not sit

Fable should not be the core football simulation model. The football intelligence should come from models trained on football event data.
