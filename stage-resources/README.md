# Fieldstate Stage Resources

This folder stores stage-by-stage inspiration, references, screenshots, notes, and resource links used to develop Fieldstate from the current evidence demo toward the full dream product.

## How to Use This Folder

- Keep one folder per development stage.
- Store inspiration as notes, links, screenshots, sketches, and product observations.
- Do not copy proprietary source code, paid datasets, private investor material, or confidential third-party content into this public docs repo.
- For each reference, record what it inspires and what Fieldstate should do differently.
- Separate current capability from future ambition. Inspiration is not a claim that Fieldstate already has the feature.

## Stage Folders

| Folder | Purpose |

| `stage-01-current-evidence-demo/` | Current real-data demo, replay, scenario editing, evidence gates. |
| `stage-02-mvp-v2-pilot-sprint/` | Pilot-ready product, saved moments, reports, walkthroughs. |
| `stage-03-data-rich-decision-intelligence/` | Licensed data, pressure, momentum, player/team intelligence. |
| `stage-04-neural-spatial-sequence-engine/` | Tracking-enriched neural/spatial modelling and simulation. |
| `stage-05-dream-product-platform/` | Full product shell, commercial workflows, platform rooms. |
| `shared-references/` | Cross-stage references and reusable design/model ideas. |

## Public-Safety Rule

This repo is public. If a reference contains commercial negotiation strategy, legal positioning, private investor comments, or non-public material, put it in the private notes repo instead and add only a public-safe pointer here.
