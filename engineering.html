Rust/open-football

Python models

Claude/Fable

Best for deep research, document refinement, factor suggestions, and structured scenario explanations.

Guardrail

Fable can propose overlooked factors and explain the model output, but it should not invent the counterfactual result. The outcome path should come from event data, statistical logic, and validation.
