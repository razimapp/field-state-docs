How to read Fieldstate's current model evidence.

The current pass-completion model is an event-only probability model. It provides useful early signal with broad held-out testing and calibration checks; it is not a finished neural football engine.

Current evidence

Evaluated, bounded, traceable

113,598 passes · 105 matches · 5 competitions.

AUC 0.7952 · Brier 0.1170 · ECE 0.0168.

Safe interpretation: credible signal for a bounded MVP use case, not universal football prediction accuracy.

The three checks

AUC

Can the model separate easier from harder passes?

AUC measures ranking power. A value of 0.5 means no useful separation; the current event-only model scores 0.7952 on the evaluated corpus.

Brier

How wrong are the probability numbers?

Brier score measures probability error. Lower is better. The current score is 0.1170, improving on the tested simple baselines.

ECE

Do predicted probabilities match observed outcomes?

Expected Calibration Error checks whether stated probabilities align with results. Lower is better. The current score is 0.0168.

Bottom line

The current model evidence supports careful continued product development. Richer 360, tracking and neural work remains a later research and funding decision until its own evidence is clean.
