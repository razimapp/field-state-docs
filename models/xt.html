Possession model

xT values ball movement into more dangerous areas.

It helps answer whether a pass or carry improved the team’s chance of creating danger.

Overview Models Data Roadmap

How it works

Split the pitch into zones. Learn how often moving the ball from one zone to another leads to danger.

Where it helps

Pass comparison, carry comparison, zone progression, and build-up analysis.

Scenario fit

Very strong for the event-level product, because it values the action before a shot happens.

How to test it

Check whether actions with higher xT lead to more shots, better territory, or more xG later.

What it misses

It is a simplified field map, so it does not fully capture pressure, player roles, or opponent shape.

Investor wording

“xT helps us explain which actions move a team closer to danger.”
